What's included
Everything you need to ship safe agents
15 Detection Rules
Pattern engine with rules covering all major AVE attack classes — from metamorphic payloads and crypto drain to jailbreak and PII exfiltration.
LLM Stage 2
Semantic analysis via LiteLLM — any provider. Catches nuanced injections that regex misses. Works with Anthropic, OpenAI, Gemini, Ollama, and 100+ more.
Watch Mode
Re-scans automatically on every file change. Run bawbel scan ./skills/ --watch and stay protected as you build.
CI/CD Ready
GitHub Actions, GitLab CI, pre-commit hooks. SARIF output uploads directly to the GitHub Security tab. Zero config, just pip install.
PiranhaDB API
Live threat intelligence API with behavioral fingerprints, IOC lists, and CVSS-AI scores for all 15 AVE records. Free and open at api.piranha.bawbel.io.
AVE Standard

Detection pipeline
Four stages. Zero false negatives.

Each engine adds a layer. All four run in sequence. Skip any stage by not installing its dependency.

Stage Engine Install Status
1a Pattern — 15 regex rules, always runs pip install bawbel-scanner Live
1b YARA — binary + text matching [yara] Live
1c Semgrep — structural pattern matching [semgrep] Live
2 LLM — semantic analysis, any provider [llm] + API key Live
3 Behavioral sandbox — Docker + eBPF runtime — v1.0 roadmap
